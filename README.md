
# Repositório de Arquivos do Projeto PDV Android - Flutter

## Comandos para Criação do Projeto Flutter

``` Dart
flutter create -i swift -a java --org br.com.sacfiscal pdvandroid_flutter
```

## Comando para limpar o projeto

``` Dart
flutter clean
```
## Comando para fazer a obter os packages do projeto

``` Dart
flutter pub get
```
## Comando para fazer a atualização dos packages do projeto

``` Dart
flutter pub upgrade
```
